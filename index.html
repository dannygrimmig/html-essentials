Getting Started

To get things started, it will be important to understand how to use your computers terminal

  1. Press cmd + space --> Type terminal into your laptops "spotlight search"

The Terminal

Your terminal is what goes on behind your computers GUI (Graphic User Interface)

  • You can navigate your desktop
  • Easily create/begin coding new fiiles
  • Communicate with GIT

Navigating The Terminal

Use the forllowing commands to begin navigating the terminal

  • pwd --> Power Working Directory (shows you your current working path)
  • ls --> list all of the items in your current directory
  • cd <directory> --> change pwd to that of the selected directory (enters that folder)
  • cd .. --> go back one directory in your pwd
  • cd ~ --> go back to home directory

Creating/Writing Files

  • mkdir <folder> --> Make Directory (creates a new folder titled folder)
  • touch <example.txt> --> creates a txt file named example.html
  • cat > <example.txt> --> creates a txt file and allows you to write it in terminal (crtr + c = exit)
  • cat <example.txt> --> views contents of your file current terminal window

Github

Setting up your branch

  • git status --> show what branch you are on, as well as if code up to date/changes
  • git checkout -b "branchname" --> creates and names a new branch
  • git checkout main --> brings you back to the main branch

Updating Git Repo w/ Changes

  • git add . --> stage everything for commit
  • git add <docname.html> --> add just this particular document
  • git commit --> commits all staged commits (from add command)
  • git push --> push your commits up to your repo
  • Pull your commits on Github
